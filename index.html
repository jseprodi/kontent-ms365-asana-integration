Microsoft 365 & Asana Integration

Initializing...

This custom app automatically syncs Kontent.ai content items to Microsoft 365 calendar events and Asana tasks.

When you open a content item in the editor, the app will:

  • Create calendar events in Microsoft 365 for assigned contributors
  • Create tasks in Asana for the content item
  • Update existing events/tasks when workflow steps change

Check the browser console for detailed logs.
